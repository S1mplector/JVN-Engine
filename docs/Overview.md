# Overview

Percept2 (JVN) is a modular 2D engine with:

- Scene graph and components for fast iteration
- Simple 2D physics with bodies and collisions
- JES DSL to author scenes, entities, input and timelines
- JavaFX-based Editor for live preview and property editing

You can author content in `.jes` and run it via the Runtime or preview it in the Editor.

Key highlights:
- Lightweight, pure-Java, JDK 17
- Swing and JavaFX rendering backends
- Extendable loader and components
